## 기아 EV 시리즈 개요

기아 EV 시리즈는 **E-GMP 플랫폼**을 기반으로 한 전기차 라인업입니다. K 시리즈(K3, K5, K8)처럼 EV+숫자 형태로 네이밍되어 있으며, 숫자가 클수록 차급이 높아집니다.

| 모델 | 차급 | 출시일 |
|------|------|--------|
| **EV3** | 소형 SUV | 2024년 5월 |
| **EV4** | 준중형 세단 | 2025년 4월 |
| **EV5** | 준중형 SUV | 2025년 9월 |
| **EV6** | 준중형 크로스오버 | 2021년 8월 |
| **EV9** | 대형 SUV | 2023년 6월 |

## 모델별 상세 분석

### EV3 - 가성비 소형 SUV

**"가장 저렴하게 기아 전기차를 만나는 방법"**

| 항목 | 내용 |
|------|------|
| 포지션 | 소형 SUV (셀토스급) |
| 가격 | 3,995만원~ |
| 주행거리 | 350km (스탠다드) / 501km (롱레인지) |
| 트렁크 | 460L |

**특징:**
- 기아 EV 시리즈 중 **가장 저렴한** 가격
- 소형이지만 **501km** 롱레인지 주행거리
- 실외 V2L 커넥터 기본 제공
- 12.3인치 내비게이션 + 기아 AI 어시스턴트

**추천 대상:**
- 출퇴근용 세컨드카
- 1~2인 가구
- 전기차 입문자

---

### EV4 - 효율 최강 세단

**"가장 멀리 가는 기아 전기차"**

| 항목 | 내용 |
|------|------|
| 포지션 | 준중형 세단 (K4 대체) |
| 가격 | 4,042만원~ |
| 주행거리 | 382km (스탠다드) / 533km (롱레인지) |
| 트렁크 | 490L |
| 공기저항계수 | 0.23Cd |

**특징:**
- 기아 전기차 중 **최장 주행거리 533km**
- 세단 특유의 낮은 공기저항으로 **최고 효율**
- 스타일리시한 패스트백 디자인
- 해치백 모델은 유럽에서만 판매

**추천 대상:**
- 장거리 출퇴근자
- 세단 스타일 선호
- 효율 중시

---

### EV5 - 패밀리 SUV

**"가족을 위한 실용적인 선택"**

| 항목 | 내용 |
|------|------|
| 포지션 | 준중형 SUV (스포티지급) |
| 가격 | 4,855만원~ |
| 주행거리 | 460km |
| 트렁크 | 513L (2열 폴딩 시 1,718L) |

**특징:**
- **정통 SUV 스타일** - 높은 전고와 넉넉한 실내
- 패밀리카에 적합한 **대용량 트렁크**
- 파노라믹 와이드 디스플레이 (12.3인치 x2 + 5인치)
- 실내 V2L 콘센트 기본

**추천 대상:**
- 자녀가 있는 가정
- SUV 스타일 선호
- 넉넉한 적재 공간 필요

---

### EV6 - 스포티 크로스오버

**"기아 전동화의 시작점, 완성도 높은 선택"**

| 항목 | 내용 |
|------|------|
| 포지션 | 준중형 크로스오버 |
| 가격 | 4,660만원~ |
| 주행거리 | 394km (스탠다드) / 494km (롱레인지) |
| 충전 | 800V 멀티 충전 시스템 |

**특징:**
- **800V/400V 멀티 충전** - 18분 만에 10→80%
- 스포티한 **크로스오버 디자인**
- 2,900mm 휠베이스로 넉넉한 뒷좌석
- **EV6 GT** - 585마력 고성능 모델 존재

**추천 대상:**
- 스포티한 디자인 선호
- 빠른 충전 속도 중시
- 고성능(GT) 옵션 고려

---

### EV9 - 플래그십 대형 SUV

**"기아 전기차의 정점"**

| 항목 | 내용 |
|------|------|
| 포지션 | 대형 SUV (팰리세이드급) |
| 가격 | 6,314만원~ |
| 주행거리 | 501km |
| 좌석 | 6인승 / 7인승 |
| 휠베이스 | 3,100mm |

**특징:**
- 기아 EV 시리즈 **최상위 모델**
- **3열 시트** - 6인승/7인승 선택
- 3,100mm 초장축 휠베이스
- 디지털 타이거 페이스 + 스타맵 LED
- **EV9 GT** - 374kW 듀얼 모터

**추천 대상:**
- 대가족 (6~7인)
- 플래그십 모델 원하는 분
- 넓은 실내 공간 최우선

## 마무리

기아 EV 시리즈는 **다양한 라이프스타일에 맞는 선택지**를 제공합니다.

**핵심 정리:**
- **가성비**: EV3 (3,995만원~)
- **효율/주행거리**: EV4 (533km)
- **패밀리 SUV**: EV5
- **스포티/충전속도**: EV6
- **플래그십/대가족**: EV9

전기차 구매 시 **주행거리, 충전 속도, 실내 공간, 예산**을 종합적으로 고려하여 본인에게 맞는 모델을 선택하시기 바랍니다.
