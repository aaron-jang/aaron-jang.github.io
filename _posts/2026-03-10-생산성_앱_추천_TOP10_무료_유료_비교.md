## 할일 관리 앱

### 1. Todoist — 크로스 플랫폼 최강

**한 줄 요약**: 어떤 기기에서든 빠르게 할일을 잡고 관리하는 앱.

> "내일 오후 3시 미팅"이라고 입력하면 날짜·시간이 자동 설정됩니다.

| 항목 | 무료 (Beginner) | 유료 (Pro) |
|------|:------:|:------:|
| 프로젝트 수 | 5개 | 300개 |
| 협업 인원 | 5명 | 25명 |
| 필터 뷰 | 3개 | 150개 |
| 캘린더 레이아웃 | X | O |
| AI 작업 분할 | X | O |
| 활동 기록 | 1주 | 무제한 |
| **가격** | **무료** | **월 ~5,800원 (연간)** |

**플랫폼**: iOS, Android, Web, Mac, Windows, Linux, Apple Watch

**장점**:
- 자연어 입력이 한국어에서도 잘 작동
- 80개 이상 연동 (Google Calendar, Slack, Zapier 등)
- 카르마 시스템으로 생산성 게이미피케이션

**단점**:
- 무료 플랜은 프로젝트 5개로 제한적
- 캘린더 뷰는 유료에서만 지원

**추천**: 회사 PC(Windows) + 개인폰(iPhone) 조합처럼 **기기가 다양한 직장인**.

| 아이젠하워 매트릭스 | X | O |
| 캘린더 뷰 | 기본 | 전체 |

- 협업 기능 없음 (개인 전용)

## 노트 앱

### 4. Notion — 올인원 작업 공간

**한 줄 요약**: 문서, 데이터베이스, 위키, 프로젝트 관리를 하나의 도구에서.

| 항목 | 무료 | 유료 (Plus) |
|------|:------:|:------:|
| 페이지 수 | 무제한 | 무제한 |
| 파일 업로드 | 5MB | 무제한 |
| 버전 기록 | 7일 | 30일 |
| 게스트 초대 | 10명 | 100명 |
| AI 응답 | 20회 (일회성) | 유료 추가 |
| **가격** | **무료** | **월 ~14,500원 (연간)** |

**플랫폼**: iOS, Android, Web, Mac, Windows

**장점**:
- 데이터베이스 뷰 전환 (테이블, 보드, 타임라인, 갤러리, 캘린더)
- 실시간 협업 편집
- 수천 개의 무료 템플릿
- 전 세계 1억 명 이상 사용 — 커뮤니티와 자료가 풍부
- 한국어 완벽 지원

**단점**:
- 오프라인 지원이 불안정
- 무료 AI는 20회 한정 (리셋 없음)
- 배우는 데 시간이 필요 (기능이 많아서)

**추천**: **팀 프로젝트가 많은 직장인·대학생**. 혼자 쓸 때도 강력하지만, 협업할 때 진가 발휘.

### 5. Obsidian — 내 데이터는 내 것

**한 줄 요약**: 로컬 마크다운 파일 기반 노트. 플러그인으로 무한 확장.

| 항목 | 무료 | 유료 (Sync) |
|------|:------:|:------:|
| 핵심 기능 | 전부 | 전부 |
| 플러그인 | 1,800+ | 1,800+ |
| 기기 간 동기화 | X (iCloud 등 수동) | O (E2E 암호화) |
| 웹 퍼블리시 | X | 별도 $8/월 |
| **가격** | **무료** | **월 ~5,800원 (연간)** |

**학생 할인**: Sync, Publish **40% 할인**

**플랫폼**: iOS, Android, Mac, Windows, Linux

**장점**:
- 모든 노트가 내 컴퓨터의 **.md 파일** — 벤더 락인 제로
- 양방향 링크 + 지식 그래프 시각화
- 1,800개 이상 커뮤니티 플러그인
- 무료 버전에서도 핵심 기능 전부 사용 가능

**단점**:
- 동기화 없이 쓰려면 iCloud/Google Drive 등 수동 설정 필요
- Notion 대비 초기 세팅이 복잡
- 실시간 협업 기능 없음

**추천**: **연구자, "세컨드 브레인" 구축자, 프라이버시 중시하는 분**. 내 데이터를 서버에 맡기고 싶지 않다면.

## 캘린더 앱

### 6. Google Calendar — 무료의 끝판왕

**한 줄 요약**: 별도 설치 없이 어디서든 되는 무료 캘린더.

**가격**: **완전 무료** (Google 계정만 있으면)

**플랫폼**: iOS, Android, Web

**장점**:
- Gmail 이메일에서 일정 자동 생성 (항공편, 호텔, 회의)
- 공유 캘린더로 팀·가족 일정 통합
- Google Meet 원클릭 화상회의 연동
- 시간대 자동 변환 (해외 미팅에 필수)

**단점**:
- 네이티브 데스크톱 앱 없음 (웹 사용)
- 디자인이 심플하지만 고급 기능 부족
- Apple Watch 앱 없음

**추천**: **캘린더에 돈 쓰고 싶지 않은 모든 사람**. 특히 Android + Gmail 사용자.

| 항목 | 내용 |
|------|------|

| Android | 무료 (프로 기능 인앱 구매) |

| **진짜 나무 기부** | **O (집중 코인으로 실제 나무 심기)** |

**플랫폼**: iOS, Android, Chrome 확장

**장점**:
- 게이미피케이션이 진짜 효과적 — 나무를 죽이기 싫어서 폰을 안 보게 됨

- iOS는 유료 (Android는 무료)
- 타이머 외 할일 관리 기능 없음

**추천**: **시험 기간 스마트폰 유혹에 약한 학생**. "나무가 죽는다"는 감정적 페널티가 의외로 강력.

## 습관 트래커 앱

### 10. Habitica — RPG로 습관 만들기

**한 줄 요약**: 습관을 지키면 레벨업, 안 지키면 HP 감소. 진짜 RPG.

| 항목 | 무료 | 유료 (구독) |
|------|:------:|:------:|
| 습관·일과·할일 | O | O |
| 파티 (팀 보스전) | O | O |
| 길드 (커뮤니티) | O | O |
| 챌린지 참여 | O | O |
| 추가 코스메틱 | X | O |
| **가격** | **무료** | **월 ~7,200원** |

**플랫폼**: iOS, Android, Web

**장점**:
- 핵심 기능이 **전부 무료**
- 3가지 작업 유형: 습관(수시), 일과(매일), 할일(일회성)
- 파티 시스템: 친구와 팀을 이루어 보스 몬스터 격파
- 습관을 안 지키면 캐릭터가 데미지를 받음 — 책임감 부여
- 커뮤니티 챌린지로 동기부여

**단점**:
- RPG에 관심 없으면 게이미피케이션이 거추장스러울 수 있음
- UI가 다소 복잡
- 유료 구독은 대부분 코스메틱 — 기능 차이는 적음

**추천**: **게임 좋아하는 학생·직장인**. 특히 혼자 습관 만들기 어려운 분에게 파티 시스템이 효과적.

> **Apple 유저 대안**: **Streaks** (7,200원 일회성) — Apple Health 연동, 최대 24개 습관 추적, 구독 없음. 깔끔한 디자인을 선호하면 Streaks, 게이미피케이션을 원하면 Habitica.

## 1주 세팅 가이드

앱을 한꺼번에 10개 깔면 3일 안에 포기합니다. **하나씩 붙이세요.**

### Day 1 — 할일 앱 세팅

1. **TickTick** 또는 **Todoist** 설치
2. 머릿속에 있는 할일 **전부** 쏟아내기 (브레인 덤프)
3. 리스트 3개만 만들기: **업무**, **개인**, **언젠가**
4. 오늘 할 일 3개만 골라서 날짜 설정

### Day 2~3 — 캘린더 정리

1. **Google Calendar** 열기
2. 캘린더 3개 분리: **업무**, **개인**, **반복 일정**
3. 반복 일정(출근, 회의, 운동) 먼저 등록
4. 할일 앱과 캘린더 연동 설정 (TickTick: 내장 / Todoist: Google Calendar 연동)

### Day 4~5 — 노트 시스템 구축

1. **Notion** 또는 **Obsidian** 설치
2. 폴더 4개만 만들기 (PARA 시스템):
   - **프로젝트**: 현재 진행 중인 것 (기한 있음)
   - **영역**: 지속적으로 관리하는 것 (건강, 재무, 커리어)
   - **자료**: 나중에 쓸 수 있는 참고 자료
   - **보관**: 완료된 프로젝트, 비활성 자료
3. 오늘 회의록이나 아이디어 하나 적어보기

### Day 6 — 집중 도구 추가

1. **Forest** 또는 **Focus To-Do** 설치
2. 뽀모도로 세션 1회 실행 (25분 집중 + 5분 휴식)
3. 가장 중요한 작업 1개에만 사용

### Day 7 — 습관 트래커 시작

1. **Habitica** 또는 **Streaks** 설치
2. 습관 **3개 이하**만 등록 (물 마시기, 10분 독서, 스트레칭 등)
3. 너무 많이 등록하면 3일 안에 포기 — **작게 시작**

## 세컨드 브레인이란?

티아고 포르테(Tiago Forte)가 제안한 개념입니다. 디지털 도구를 활용해 **정보를 수집하고 정리하여 필요할 때 꺼내 쓰는 시스템**입니다.

### CODE 프레임워크

| 단계 | 의미 | 도구 예시 |
|------|------|----------|
| **C**apture (수집) | 인상 깊은 것은 무엇이든 저장 | Notion 웹 클리퍼, Obsidian |
| **O**rganize (정리) | PARA 시스템으로 분류 | Notion 데이터베이스, Obsidian 폴더 |
| **D**istill (정제) | 핵심만 추출, 요약 | 하이라이트, 한 줄 요약 |
| **E**xpress (표현) | 수집한 지식으로 결과물 만들기 | 블로그, 보고서, 프레젠테이션 |

완벽한 시스템을 만들려다 시간을 다 쓰면 본말전도입니다. **쓰면서 다듬는 게 핵심**입니다.
