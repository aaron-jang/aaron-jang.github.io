### 6. AWTRIX 설치 및 설정

Ulanzi TC001은 공장 출하 시 AWTRIX Light가 설치되어 나오지만, 최신 버전으로 업데이트하거나 처음 설정하는 방법을 알려드립니다.

**📱 초기 설정 (3분 완료)**

1. **전원 연결**
   - USB Type-C 케이블로 전원 연결
   - 첫 부팅 시 "AWTRIX_XXXX" WiFi 신호 발생

2. **WiFi 연결**
   - 스마트폰으로 "AWTRIX_XXXX" WiFi에 연결
   - 자동으로 설정 페이지 열림 (또는 `192.168.4.1` 접속)
   - 집 WiFi 이름/비밀번호 입력

3. **웹 인터페이스 접근**
   - 시계 화면에 표시되는 IP 주소 확인
   - 브라우저에서 해당 IP 주소 입력 (예: `192.168.0.100`)
   - AWTRIX Light 대시보드 접속 완료!

**🔧 펌웨어 업데이트**

AWTRIX Light는 지속적으로 업데이트되므로 최신 기능을 사용하려면 업데이트를 권장합니다.

```
1. 웹 인터페이스 접속
2. Settings → Update 메뉴
3. "Check for Updates" 클릭
4. 업데이트 있으면 "Update" 버튼 클릭
5. 자동 재부팅 후 완료
```

**💡 유용한 설정**

- **밝기 조절**: Settings → Display → Brightness (낮/밤 자동 조절 가능)
- **시간대 설정**: Settings → Time → Timezone (Asia/Seoul)
- **자동 밝기**: 주변 조도에 따라 자동 밝기 조절
- **전환 효과**: 화면 전환 시 애니메이션 효과 설정

### 8. Home Assistant 연동

스마트홈 시스템을 사용 중이라면 AWTRIX와 Home Assistant를 연동하여 더욱 강력한 기능을 활용할 수 있습니다.

**🏠 Home Assistant란?**

오픈소스 스마트홈 플랫폼으로, 다양한 스마트 기기들을 하나로 통합하여 제어할 수 있습니다.

**🔗 연동 방법**

1. **MQTT 브로커 설정**
   ```yaml
   # AWTRIX Settings → MQTT
   Broker: [Home Assistant IP]
   Port: 1883
   Username: [MQTT 사용자]
   Password: [MQTT 비밀번호]
   ```

2. **Home Assistant 설정**
   ```yaml
   # configuration.yaml
   mqtt:
     sensor:
       - name: "AWTRIX Display"
         state_topic: "awtrix/stats"
   ```

