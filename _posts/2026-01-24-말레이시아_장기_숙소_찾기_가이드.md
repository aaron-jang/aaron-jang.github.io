## 숙소 찾기 타임라인

장기 숙소는 **현지에서 직접 보고 계약**하는 것을 권장합니다.

```
한국에서: 임시 숙소 예약 (호텔/에어비앤비, 2~4주)
    ↓
도착 후 1주일: 매물 탐색, 여러 곳 방문
    ↓
도착 후 2~3주: 계약 체결
    ↓
도착 후 3~4주: 입주
```

> 💡 **Tip**: 온라인으로 미리 계약하면 실제와 다른 경우가 많습니다. 반드시 직접 방문 후 결정하세요.

## 매물 찾는 방법

### 1. 온라인 포털

| 사이트 | 특징 | 링크 |
|--------|------|------|
| **PropertyGuru** | 말레이시아 최대 부동산 포털 | [propertyguru.com.my](https://www.propertyguru.com.my) |
| **iProperty** | 다양한 매물, 상세 정보 | [iproperty.com.my](https://www.iproperty.com.my) |
| **EdgeProp** | 시장 분석 정보 풍부 | [edgeprop.my](https://www.edgeprop.my) |

**검색 팁:**
- "Fully Furnished" 필터 사용
- 지역명 + "condo for rent" 검색
- 최근 등록 매물 우선 확인

### 2. 부동산 에이전트

현지 에이전트를 통하면 협상과 계약이 수월합니다.

**에이전트 수수료:**
- 보통 월세 1개월분
- 일반적으로 집주인이 부담하지만, **지역/계약에 따라 다를 수 있음**
- 계약 전 수수료 부담 주체를 반드시 확인하세요

**에이전트 찾는 방법:**
- PropertyGuru/iProperty에서 매물 문의 시 연결
- 한인 커뮤니티 추천
- 학교 학부모 네트워크

> ⚠️ **주의**: 2+1 계약을 유도하는 에이전트는 피하세요 (수수료 목적). 1+1이 표준입니다.

### 3. 한인 커뮤니티

| 채널 | 특징 |
|------|------|
| 카카오톡 단톡방 | 지역별 맘카페, 학부모 그룹 |
| 네이버 카페 | 말레이시아 유학/이민 카페 |
| 페이스북 | Koreans in KL 등 |

한인 커뮤니티에서는 **직거래 매물**이나 **이전 거주자 후기**를 얻을 수 있습니다.

## 계약 절차

### 계약 단계

```
1. 매물 방문 및 선택
    ↓
2. Letter of Offer (가계약서) 작성
   - 계약금: 월세 1개월분
   - Stamping Fee: 월세의 약 15%
    ↓
3. 조건 협상 (수리, 가구 추가 등)
    ↓
4. Tenancy Agreement (본계약서) 작성
    ↓
5. 잔금 납부 및 입주
```

### 비용 구조

| 항목 | 금액 | 시기 |
|------|------|------|
| **보증금 (Deposit)** | 월세 2개월분 | 계약 시 |
| **유틸리티 보증금** | 월세 0.5개월분 | 계약 시 |
| **Stamping Fee** | 월세의 약 15% | 계약 시 |
| **첫 달 월세** | 월세 1개월분 | 입주 시 |

**예시: 월세 RM 3,000 콘도**

| 항목 | 금액 |
|------|------|
| 보증금 (2개월) | RM 6,000 |
| 유틸리티 보증금 (0.5개월) | RM 1,500 |
| Stamping Fee (15%) | RM 450 |
| 첫 달 월세 | RM 3,000 |
| **합계** | **RM 10,950 (약 330만원)** |

> 💡 보증금은 퇴거 시 하자가 없으면 **환불**됩니다.

### 계약 기간

| 기간 | 특징 |
|------|------|
| **1년** | 표준, 대부분 선택 |
| **2년** | 월세 협상 유리 (5~10% 할인 가능) |
| **6개월 미만** | 거의 없음, 있어도 비쌈 |

### 계약서 확인 사항

반드시 확인하고 서명하세요:

- [ ] **월세 금액** 정확한지
- [ ] **보증금 환불 조건**
- [ ] **관리비 부담** (보통 집주인)
- [ ] **중도 해지 조건** (보통 보증금 몰수)
- [ ] **퇴거 통보 기한** (보통 1~2개월 전)
- [ ] **수리 비용 부담** (소액: 세입자, 고액: 집주인)
- [ ] **반려동물 허용 여부**

## 협상 팁

### 협상 가능한 것

| 항목 | 협상 방법 |
|------|----------|
| **월세** | 장기 계약 (2년) 제안, 즉시 입주 가능 어필 |
| **가구/가전 추가** | 세탁기, 건조기, 침대 매트리스 교체 요청 |
| **수리** | 입주 전 에어컨 청소, 페인트, 하자 수리 요청 |
| **관리비** | 집주인 부담 명시 요청 |

### 협상이 어려운 것

- 보증금 금액 (2개월이 표준)
- Stamping Fee (법정 비용)
- 계약 기간 1년 미만

### 협상 예시

> "2년 계약으로 할 테니 월세를 RM 2,800으로 해주실 수 있나요?"
>
> "에어컨이 오래됐는데, 입주 전에 청소해주시면 바로 계약하겠습니다."

## 입주 후 할 일

### 입주 당일

1. **하자 점검 및 사진 촬영**
   - 모든 방, 가구, 가전 상태 사진
   - 기존 흠집, 얼룩 기록
   - 에이전트/집주인에게 공유

2. **작동 확인**
   - 에어컨 전체
   - 온수기
   - 변기 물내림
   - 수도 수압
   - 전등

### 입주 후 1주일

- [ ] 전기/수도 명의 확인 (집주인 or 본인)
- [ ] 인터넷 설치 신청 (Maxis, Time, Unifi 등)
- [ ] 관리사무소 등록
- [ ] 주차 카드/스티커 발급
- [ ] 쓰레기 분리수거 방법 확인

### 하자 보수 기간

입주 후 **1개월**은 하자 보수 기간입니다.
- 발견된 문제는 즉시 에이전트/집주인에게 연락
- 사진과 함께 문서로 기록
- 보통 **RM 300~500 이하**는 세입자 부담

## 주의사항

### 허위 매물 피하기

| 경고 신호 | 대처 |
|----------|------|
| 터무니없이 저렴한 가격 | 시세 확인 후 비교 |
| 가상 인테리어 사진 | 실제 사진 요청 |
| 보증금 먼저 송금 요구 | 직접 방문 전 송금 금지 |
| 집주인 직접 만남 불가 | 에이전트 통해 확인 |

### 계약 전 확인

- 집주인 신분증 확인 (IC 카드)
- 에이전트 라이센스 확인
- 부동산 등기 확인 (가능하면)

### 퇴거 시

- 퇴거 **1~2개월 전** 서면 통보
- 원상복구 (못 구멍, 벽지 등)
- 공과금 정산
- 보증금 환불 일정 확인 (보통 14~30일)
