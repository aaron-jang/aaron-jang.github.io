## 3. 맥북 네오 — 핵심 포인트 5가지

### A18 Pro 칩 — 아이폰 칩이 맥북에

맥북 역사상 처음으로 **아이폰 칩**이 들어갑니다. iPhone 16 Pro에 탑재된 A18 Pro입니다.

- **단일 코어**: M1 대비 **40~50% 빠름** (Geekbench 6 기준 ~3,450점)
- **멀티 코어**: M1~M2급 성능 (~8,500점)
- 일반 PC 대비 **일상 작업 50% 빠름**, **AI 작업 3배**, **사진 편집 2배** (Apple 공식)

웹 브라우징, 문서 작업, 영상 시청, 가벼운 사진·영상 편집까지 충분한 성능입니다.

### Apple Intelligence 지원

16코어 Neural Engine으로 **Apple Intelligence 전체 기능**을 지원합니다.

- Writing Tools (글쓰기 도구)
- Image Playground (이미지 생성)
- Genmoji (맞춤 이모지)
- Siri 강화 (화면 인식, 자연어 명령)
- 알림·메일 요약

99만 원 맥북에서 AI 기능을 풀로 쓸 수 있습니다.

![MacBook Neo — New to Mac](/assets/images/macbook_neo_ntm.jpg)

### 4가지 컬러 — 가장 컬러풀한 맥북

| 색상 | 설명 |
|------|------|
| **실버** | 클래식 실버 |
| **블러시** | 옅은 핑크 |
| **시트러스** | 밝은 옐로 |
| **인디고** | 짙은 블루 |

키보드와 트랙패드까지 **색상 매칭**. 2021년 iMac 이후 맥에서 가장 다양한 색상 라인업입니다.

### 팬리스 설계 — 완전 무소음

A18 Pro의 전력 효율 덕분에 **팬이 없습니다**. 어떤 작업을 해도 소음이 0입니다. 도서관, 카페, 강의실에서 완벽합니다.

### 16시간 배터리

36.5Wh 배터리로 영상 재생 **최대 16시간**. MacBook Air(18시간)보다 약간 짧지만, 하루 종일 충전 없이 사용 가능합니다.

## 4. 아쉬운 점

### 8GB RAM — 업그레이드 불가

모든 모델이 **8GB 고정**입니다. MacBook Air M5는 16GB부터 시작하는데, Neo는 8GB 옵션만 존재합니다.

웹 브라우징 + 문서 작업 수준에서는 충분하지만, 탭을 많이 열거나 무거운 앱을 동시에 돌리면 부족할 수 있습니다.

### MagSafe 없음

충전은 **USB-C 포트**로만 가능합니다. MagSafe 충전기를 따로 살 수 없습니다. 20W USB-C 어댑터가 기본 포함입니다.

### USB 포트 — 왼쪽에만 2개

USB-C 포트 2개가 **왼쪽에만** 있습니다. 하나는 USB 3(10Gb/s), 다른 하나는 **USB 2(480Mb/s)**입니다. 외장 디스플레이는 USB 3 포트에서만 연결 가능합니다. 오른쪽에는 3.5mm 헤드폰 잭만 있습니다.

### Touch ID — 256GB 모델에 없음

99만 원 기본 모델에는 **Touch ID가 없습니다**. 지문 인식이 필요하면 119만 원 512GB 모델을 선택해야 합니다.

### 60Hz 디스플레이

MacBook Air와 동일하게 60Hz입니다. ProMotion 120Hz는 MacBook Pro에서만 가능합니다.

## 5. MacBook Neo vs MacBook Air M5

| 항목 | MacBook Neo | MacBook Air M5 (13인치) |
|------|------------|------------------------|
| **가격** | **990,000원~** | 1,790,000원~ |
| **칩셋** | A18 Pro | M5 |
| **CPU** | 6코어 | 10코어 |
| **GPU** | 5코어 | 10코어 |
| **RAM** | 8GB | **16GB** |
| **기본 용량** | 256GB | **512GB** |
| **디스플레이** | 13인치 60Hz LCD | 13.6인치 60Hz LCD |
| **배터리** | 16시간 | **18시간** |
| **무게** | 1.23kg | 1.24kg |
| **MagSafe** | ❌ | ✅ |
| **Touch ID** | 512GB만 | ✅ |
| **포트** | USB-C ×2 왼쪽 (USB 3 + USB 2) | USB-C ×2 + MagSafe |
| **외부 디스플레이** | 1대 (좌측만) | **최대 2대** |
| **Thunderbolt** | ❌ | ❌ |
| **Wi-Fi** | 6E | **7** |
| **색상** | 4색 | 4색 |

### 가격 차이 80만 원, 뭐가 다른가?

MacBook Air M5는 **M5 칩(CPU·GPU 약 2배)**, **RAM 2배(16GB)**, **용량 2배(512GB)**, **MagSafe**, **Wi-Fi 7**을 갖고 있습니다.

하지만 웹 브라우징, 문서 작업, 영상 시청 등 **일상 사용**에서 체감 차이는 크지 않습니다. Neo의 A18 Pro도 이런 작업에는 충분합니다.

### A18 Pro vs M5 — 칩 성능 비교

| 항목 | A18 Pro (Neo) | M5 (Air) |
|------|--------------|----------|
| **아키텍처** | ARMv9 모바일 설계 | ARMv9 데스크톱급 설계 |
| **공정** | 3nm (TSMC 2세대) | 3nm (TSMC 3세대) |
| **트랜지스터** | ~200억 개 (추정) | ~250억 개 이상 (추정) |
| **CPU 코어** | 6코어 (2성능 + 4효율) | 10코어 (4성능 + 6효율) |
| **CPU 클럭** | 성능 4.0GHz / 효율 2.4GHz | 성능 4.6GHz / 효율 3.0GHz |
| **GPU 코어** | 5코어 (iPhone 16 Pro는 6코어) | 10코어 |
| **레이트레이싱** | 1세대 | 3세대 + 신경 가속기 |
| **Neural Engine** | 16코어 (35 TOPS) | 16코어 + GPU별 신경 가속기 |
| **RAM** | 8GB | 최대 32GB |
| **메모리 대역폭** | 60GB/s | 153GB/s |
| **Geekbench 6 싱글** | ~3,460점 | **~4,100점** (+18%) |
| **Geekbench 6 멀티** | ~8,600점 | **~15,400점** (+80%) |
| **Geekbench 6 GPU** | ~32,000점 | **~74,000점** (+130%) |
| **최대 전력** | 8~10W (팬리스) | 20~25W (팬리스) |

**핵심 차이**: M5는 CPU 멀티코어 약 **80% 빠르고**, GPU는 **2배 이상** 빠릅니다. 메모리 대역폭도 2.5배.

하지만 **싱글코어**는 18% 차이뿐이라, 웹 브라우징·문서·메일 같은 **일상 작업에서 체감 차이는 크지 않습니다**.

**A18 Pro로 충분한 작업**:
- 웹 브라우징, 문서 작업, 메일
- 영상 시청, 음악 스트리밍
- 가벼운 사진 편집, 기본 영상 편집
- Apple Intelligence AI 기능
- 코딩 (가벼운 프로젝트)

**M5가 필요한 작업**:
- 대량 RAW 사진 처리, 고급 편집
- 4K/8K 영상 편집, 멀티캠 타임라인
- 3D 모델링·렌더링 (Blender, Cinema 4D)
- 로컬 AI 모델 실행 (LLM, 이미지 생성)
- 무거운 멀티태스킹 (탭 수십 개 + IDE + 시뮬레이터)

## 6. 누가 사야 할까?

### 추천

- **첫 맥북** 구매자 — 99만 원에 macOS + Apple Intelligence 입문
- **학생** — 교육 할인 시 더 저렴, 팬리스로 강의실에서 무소음
- **자녀·부모님용** — 웹·메일·영상 위주 가벼운 사용
- **서브 노트북** — 메인은 데스크탑, 외출용 가벼운 맥
- **크롬북·윈도우 저가 노트북** 대신 macOS를 원하는 분

### 비추천

- **RAM 16GB 이상** 필요한 분 → MacBook Air M5 추천
- **영상 편집·개발** 등 무거운 작업 → MacBook Air 또는 Pro 추천
- **MagSafe 충전** 필수인 분 → MacBook Air 추천
- **외부 모니터 2대** 연결이 필요한 분 → MacBook Air 추천

## 7. 마치며

맥북 네오의 핵심은 **"99만 원에 Apple Silicon + Apple Intelligence + macOS"**입니다.

A18 Pro 칩은 일상 작업에 충분한 성능을 내고, 팬리스 설계에 16시간 배터리, 4가지 컬러까지. 8GB RAM과 MagSafe 미지원이 아쉽지만, 이 가격대에서 macOS 경험을 제공한다는 것 자체가 Apple 역사상 처음입니다.

사전 주문은 **3월 6일 오전 10시**, 출시는 **3월 11일**입니다.

### 맥북 네오 가격은 얼마인가요?

MacBook Neo는 256GB 모델 990,000원, 512GB(Touch ID 포함) 모델 1,190,000원입니다. Apple 역사상 가장 저렴한 맥북입니다.

### 맥북 네오는 언제 출시되나요?
