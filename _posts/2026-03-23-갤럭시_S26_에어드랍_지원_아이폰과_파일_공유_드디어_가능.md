안녕하세요. 수수입니다.

"갤럭시에서 아이폰으로 사진 보내려면 카카오톡밖에 없나?" — 이런 불편함, 드디어 끝납니다.

**삼성전자가 오늘(3월 23일) 갤럭시 S26 시리즈에 애플 에어드랍(AirDrop) 연동 기능을 공식 배포**하기 시작했습니다. 퀵쉐어(Quick Share)를 통해 아이폰·아이패드·맥과 별도 앱 설치 없이 파일을 주고받을 수 있게 된 것이죠.

오늘은 이 기능의 작동 원리, 설정 방법, 지원 기기, 그리고 주의사항까지 한 번에 정리합니다.

## 30초 핵심 요약

- **대상 기기**: 갤럭시 S26 / S26+ / S26 울트라 (추후 다른 갤럭시로 확대)
- **작동 방식**: 퀵쉐어(Quick Share)가 애플 에어드랍 프로토콜을 지원
- **별도 앱 불필요**: 갤럭시 ↔ 아이폰·아이패드·맥 간 바로 전송
- **출시일**: 2026년 3월 23일 한국 시작, 이후 글로벌 순차 확대
- **설정**: 퀵쉐어 설정에서 "Apple 기기와 공유" 옵션 활성화 필요

## 왜 이게 큰 뉴스인가?

### 오랜 숙제: 안드로이드와 아이폰 사이의 벽

그동안 안드로이드와 아이폰 사이 파일 전송은 정말 불편했습니다.

| 방법 | 문제점 |
|------|--------|
| 카카오톡·메신저 | 사진 화질 저하, 용량 제한 |
| 이메일 | 대용량 파일 불가, 번거로움 |
| 클라우드(구글 드라이브 등) | 업로드-다운로드 2단계 필요 |
| USB 케이블 | 케이블 필요, PC 경유해야 하는 경우도 |
| 서드파티 앱(Send Anywhere 등) | 양쪽 모두 앱 설치 필요 |

아이폰끼리는 에어드랍, 갤럭시끼리는 퀵쉐어로 간편하게 보낼 수 있었지만, **두 진영 사이에는 깔끔한 해결책이 없었습니다**.

### 구글 픽셀이 먼저 뚫었다

2025년 11월, 구글이 **픽셀 10 시리즈**에서 퀵쉐어를 통한 에어드랍 호환을 최초로 선보였습니다. 이후 픽셀 9 시리즈까지 확대되었고, 이제 삼성이 갤럭시 S26으로 합류하면서 **주류 안드로이드 기기에서도 에어드랍이 가능한 시대**가 열린 것입니다.

---

## 어떻게 작동하나?

### 기술 배경

구글이 퀵쉐어(Quick Share)에 애플의 에어드랍 프로토콜을 지원하도록 업데이트했습니다. 블루투스로 주변 기기를 탐색하고, Wi-Fi Direct 또는 로컬 네트워크를 통해 고속 전송하는 방식입니다.

### 전송 흐름 (갤럭시 → 아이폰)

1. 갤럭시에서 사진·파일 선택 → **공유 → 퀵쉐어** 선택
2. 주변 아이폰이 목록에 표시됨
3. 아이폰에서 **수락** 누르면 전송 완료

### 전송 흐름 (아이폰 → 갤럭시)

1. 아이폰에서 사진·파일 선택 → **공유 → AirDrop** 선택
2. 주변 갤럭시 S26이 에어드랍 목록에 표시됨
3. 갤럭시에서 **수락** 누르면 전송 완료

> 양쪽 모두 블루투스와 Wi-Fi가 켜져 있어야 합니다.

## 설정 방법

### 갤럭시 S26 설정

1. **설정** → **연결된 기기** 또는 **빠른 공유(Quick Share)** 진입
2. **"Apple 기기와 공유"** 옵션을 **활성화**
3. 공유 대상을 "연락처만" 또는 "모든 사람"으로 설정

> ⚠️ 에어드랍 연동은 기본 **비활성화** 상태입니다. 반드시 수동으로 켜야 합니다.

### 아이폰 설정

1. **설정** → **일반** → **AirDrop** 진입
2. 수신 대상을 **"모든 사람에 대해 10분 동안"** 으로 설정
3. 블루투스·Wi-Fi가 켜져 있는지 확인

> 아이폰 쪽은 "연락처만"이 아닌 **"모든 사람"**으로 설정해야 안드로이드 기기가 보입니다. (연락처 기반 필터링은 Apple 기기끼리만 작동)

## 주의사항 & 팁

### 꼭 알아둘 것

- **소프트웨어 업데이트 필수**: 퀵쉐어 최신 버전으로 업데이트해야 에어드랍 연동이 활성화됩니다.
- **아이폰 수신 설정**: "모든 사람에 대해 10분 동안"을 선택해야 합니다. 10분이 지나면 자동으로 "연락처만"으로 돌아가므로, 전송 전에 다시 확인하세요.
- **거리 제한**: 블루투스 기반이므로 약 **9미터(30피트)** 이내에서 사용하세요.
- **대용량 전송**: Wi-Fi Direct를 활용하므로 큰 파일도 빠르게 전송 가능합니다.

### 이럴 때 안 되면?

| 증상 | 해결 방법 |
|------|-----------|
| 상대 기기가 안 보임 | 블루투스·Wi-Fi 양쪽 모두 켜져 있는지 확인 |
| 아이폰에서 갤럭시 안 보임 | 갤럭시 퀵쉐어 → "Apple 기기와 공유" 활성화 확인 |
| 갤럭시에서 아이폰 안 보임 | 아이폰 에어드랍 → "모든 사람에 대해 10분 동안" 설정 확인 |
| 전송 실패 | 양쪽 기기를 가까이 두고, 퀵쉐어 앱 최신 업데이트 확인 |
| 기능 자체가 없음 | 갤럭시 소프트웨어 업데이트 확인 (설정 → 소프트웨어 업데이트) |
