안녕하세요. 수수입니다.

2025년 6월, 충격적인 인수 소식이 전해졌습니다.

**31세 이스라엘 개발자**가 혼자서 만든 AI 앱 빌더가, 출시 **6개월 만에 8,000만 달러(약 1,100억 원)**에 팔렸습니다.

투자 한 푼 안 받고, 직원도 없이, 사이드 프로젝트로 시작한 서비스가 1,100억 원짜리 회사가 된 것입니다.

이것이 **Base44**와 창업자 **마오르 슬로모(Maor Shlomo)**의 이야기입니다.

## 2. 마오르 슬로모는 누구인가?

### 경력

| 시기 | 내용 |
|------|------|
| **군 복무** | 이스라엘 정보부대 8200 (엘리트 사이버 부대) |
| **첫 창업** | Explorium (데이터 발견 플랫폼) |
| **Explorium 성과** | 1억 2,700만 달러 투자 유치, Forbes 30 Under 30 |
| **2024년** | 예비군 복무 (10·7 전쟁) |
| **2025년 2월** | Base44 출시 |

### 창업 계기: 여자친구 홈페이지에서 시작

2024년 가을, 슬로모는 **아티스트인 여자친구의 소규모 비즈니스 홈페이지**를 워드프레스로 만들어주려다 큰 좌절을 겪었습니다.

> "정말 고통스러웠어요. 드래그 앤 드롭 인터페이스는 모바일에서 깨지고, 데이터 관리는 복잡하고... LLM이 이 코드를 짤 수 있다는 건 알았지만, 데이터베이스 연결이나 클라우드 배포를 처리할 인프라가 없었습니다."

동시에 이스라엘 스카우트 단체의 소프트웨어 제작도 도와주고 있었는데, 두 경우 모두 같은 생각이 들었습니다:

> **"LLM이 이걸 만들 수 있어야 하는데?"**

그래서 직접 그 인프라를 만들기로 했습니다. 이것이 Base44의 시작이었습니다.

### Claude가 바꾼 게임

처음에는 GPT-4 API를 사용했지만, 모호한 아이디어를 동작하는 소프트웨어로 변환하기엔 부족했습니다. 그런데 2024년 10월, **Anthropic이 Claude 3.5 Sonnet 새 버전**을 출시하자 모든 것이 바뀌었습니다.

슬로모는 OpenAI에서 Anthropic API로 전환했고, 프롬프트로 만든 앱이 **"살아 움직이기 시작"**했다고 합니다.

필리핀과 태국을 여행하면서 첫 버전을 완성하고, 2025년 2월에 출시했습니다.

### Base44라는 이름

> "소프트웨어를 만드는 '기반(Base)'이라는 느낌을 원했어요. Base.com은 없어서, 제 생일이 2월 2일이라 2를 두 번 곱해 44로 했죠. Base64라는 인코딩 표준처럼 들리니까 괜찮았어요. 자연어를 소프트웨어로 인코딩한다는 의미로요."

## 3. AI가 나의 팀이었다

### "AI 팀을 관리하고 있었을 뿐"

슬로모는 1인 개발자 경험을 이렇게 설명했습니다:

> "혼자라고 해도, 사실 **AI로 구성된 팀이 코드를 작성하는 것을 관리**하는 겁니다. 지난 3개월간 HTML이나 JavaScript를 한 줄도 직접 작성하지 않았어요."

Base44 코드의 **90%는 Claude(Anthropic)가 작성**했습니다. 한 기사는 이를 이렇게 표현했습니다:

> **"Claude는 단순한 코드 생성기가 아니었다 — 그의 공동창업자였다."**

### AI 모델별 역할 분담

슬로모는 하나의 AI만 쓰지 않았습니다. **업무별로 최적의 AI 모델**을 배치했습니다.

| AI 모델 | 역할 |
|---------|------|
| **Claude (Anthropic)** | 앱 생성, UI 코드, 메인 개발 (90%) |
| **Gemini (Google)** | 복잡한 버그 수정, 로직 처리 |
| **GPT-5 (OpenAI)** | 특정 태스크 보조 |

> "Anthropic의 Claude가 코딩에는 여전히 압도적으로 최고의 AI 모델이지만, Gemini나 GPT-5가 더 나은 작업도 있어요. **모델을 빠르게 섞어 쓰는 것**이 우리의 가장 큰 강점입니다."

### 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| **백엔드** | Python | AI가 잘 작성하는 언어 |
| **프론트엔드** | JSX (TypeScript X) | **TypeScript 제거** → AI가 코드 생성하기 쉽게 |
| **DB** | MongoDB | 유연한 스키마, AI 친화적 |
| **호스팅** | Render.com | 간편한 배포 |
| **코드 구조** | 단일 레포 | AI에게 더 나은 컨텍스트 제공 |

**핵심 설계 철학**: AI가 잘하는 것에 맞춰 인프라를 설계했습니다. AI에게 복잡한 지시를 강요하는 대신, **AI의 행동 패턴을 연구하고 플랫폼을 그에 맞게 정렬**했습니다.

### 생산성 극대화

슬로모는 심한 **ADHD**가 있어서, 집중을 위한 환경 최적화에도 신경 썼습니다.

- **RescueTime**으로 산만함 차단
- Base44로 자체 콘텐츠/게시물 관리 앱 제작
- 딥 워크 환경 최적화

### 첫 채용은 매각 1.5개월 전

| 시기 | 인원 |
|------|------|
| 2025년 2~4월 | **1명** (완전 솔로) |
| 매각 1.5개월 전 | 첫 직원 채용 시작 |
| 매각 시점 (6월) | 8명 |

매각 시점에 8명이었지만, **대부분의 여정은 혼자**였습니다. 이 8명은 인수 후 **2,500만 달러의 잔류 보너스**를 받게 됩니다.

## 4. Base44란?

### 한 줄 설명

**대화만으로 웹사이트와 앱을 만드는 AI 플랫폼.**

코딩을 전혀 몰라도, 원하는 것을 말로 설명하면 AI가 완성된 소프트웨어를 만들어줍니다.

### 기존 방식 vs Base44

| 항목 | 기존 방식 | Base44 |
|------|----------|--------|
| **필요 기술** | 코딩, 디자인, 서버 관리 | 없음 (대화만) |
| **개발 기간** | 수주~수개월 | 수분~수시간 |
| **비용** | 수백만~수천만원 | 월 20~200달러 |
| **포함 기능** | 별도 구현 필요 | DB, 인증, 배포 자동 |

### 핵심 기능

**자동으로 처리되는 것들:**
- 데이터베이스 설계 및 구축
- 사용자 인증 (로그인/회원가입)
- 백엔드 로직
- 호스팅 및 배포
- 반응형 디자인

사용자는 **"이런 앱 만들어줘"**라고 말하기만 하면 됩니다.

### 바이브 코딩 (Vibe Coding)

Base44가 출시된 2025년 2월, OpenAI 공동창업자 **안드레이 카르파시(Andrej Karpathy)**가 이런 방식의 개발을 **"바이브 코딩(Vibe Coding)"**이라고 명명했습니다.

> 코드를 직접 작성하는 대신, 원하는 "분위기(vibe)"를 설명하면 AI가 코드를 만드는 것.

Base44는 이 바이브 코딩 트렌드의 대표적인 서비스가 되었습니다.

## 4. 폭발적 성장

### 성장 타임라인

| 시기 | 이벤트 |
|------|--------|
| **2025년 2월** | Base44 출시 |
| **출시 3주 후** | **ARR 100만 달러** 달성 |
| **2025년 3월** | 월 매출 150만 달러 |
| **2025년 5월** | 월 이익 18만 9천 달러, 30만 사용자 |
| **2025년 6월** | **Wix에 8,000만 달러 매각** |
| **2025년 3분기** | 200만 사용자 (인수 후 7배 성장) |
| **2025년 말** | ARR 5,000만 달러 전망 |

### 성장 비결

**1) 빌드 인 퍼블릭 (Build in Public)**
- SNS에서 매출, 사용자 수, 기술적 도전을 공개적으로 공유
- 마케팅 예산 **0원**

**2) 제품 자체의 힘**
- 진짜로 코딩 없이 앱을 만들 수 있었음
- DB, 인증, 배포까지 자동 처리
- 경쟁사 대비 진입 장벽이 가장 낮음

**3) 타이밍**
- 바이브 코딩이라는 용어가 등장한 바로 그 시점에 출시
- AI 앱 빌더에 대한 수요가 폭발하는 시기

**에피소드**: 슬로모는 여자친구에게 "2025년 말까지 ARR 150만 달러 찍으면 좋은 차 사자"고 했는데, **4주 만에** 달성했습니다.

| 단계 | 내용 |

마오르 슬로모는 이렇게 회상했습니다:

> "변호사들이 목요일 밤에 계약을 마무리했고, 금요일 아침에 서명하기로 했어요. 바로 그 금요일 아침에 이란과의 전쟁이 터졌습니다."

## 9. 마무리

### 핵심 메시지

| 항목 | Base44 사례 |
|------|------------|
| **창업자** | 1명 (마오르 슬로모) |
| **투자금** | 0원 |

### 시사점

Base44의 이야기가 던지는 메시지는 명확합니다:

1. **아이디어와 실행력이 자본보다 중요한 시대**가 왔습니다
2. **AI 도구가 1인 개발자의 생산성을 100배** 끌어올렸습니다
3. **바이브 코딩**은 일시적 유행이 아닌, 소프트웨어 개발의 새로운 패러다임입니다
4. **"코딩을 못해도 앱을 만든다"**는 것이 현실이 되었습니다

Sam Altman이 예언한 **"직원 1명짜리 10억 달러 기업"**의 시대가 시작되고 있습니다.
