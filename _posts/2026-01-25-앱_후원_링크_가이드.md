## 💡 후원 링크를 추가해야 하는 이유

| **커뮤니티 형성** | 후원자와 더 깊은 관계 구축 |

- ✅ 좋아하는 앱의 개발 지속을 지원
- ✅ 개발자와 직접 소통할 기회

## 🌍 글로벌 후원 플랫폼

### 1. Ko-fi - 수수료 최저, 추천!

<a href="https://ko-fi.com" target="_blank">Ko-fi 바로가기</a>

**Ko-fi**는 수수료가 가장 낮은 글로벌 후원 플랫폼입니다.

**장점**:
- ✅ **0% 플랫폼 수수료** (무료 플랜)
- ✅ 회원가입 없이 바로 결제 가능
- ✅ PayPal, 신용카드, Google Pay, Apple Pay 지원
- ✅ 일회성 후원 + 멤버십 모두 지원
- ✅ 디지털 상품 판매 가능

**단점**:
- ❌ 한국어 미지원
- ❌ PayPal 수령 시 환전 수수료 발생

**수수료 구조**:
```
무료 플랜: 0% + 결제수수료 (~3%)
Ko-fi Gold ($12/월): 0% + 결제수수료 + 프리미엄 기능
```

**후원 링크 예시**:
```
https://ko-fi.com/내아이디
```

---

### 2. Buy Me a Coffee - 간편한 설정

<a href="https://buymeacoffee.com" target="_blank">Buy Me a Coffee 바로가기</a>

**Buy Me a Coffee**는 설정이 간편하고 직관적인 UI를 제공합니다.

**장점**:
- ✅ 깔끔하고 직관적인 인터페이스
- ✅ 회원가입 없이 결제 가능
- ✅ Stripe 통한 Payout (한국 지원)
- ✅ 멤버십, 커미션 기능 지원

**단점**:
- ❌ 5% 플랫폼 수수료
- ❌ PayPal 미지원
- ❌ 총 수수료 ~9%로 높은 편

**수수료 구조**:
```
플랫폼 수수료 5% + 결제수수료 3% + $0.30
= 약 9% (거래당)
```

**후원 링크 예시**:
```
https://buymeacoffee.com/내아이디
```

---

### 3. Patreon - 정기 후원자 관리

<a href="https://patreon.com" target="_blank">Patreon 바로가기</a>

**Patreon**은 정기 멤버십에 특화된 플랫폼입니다.

**장점**:
- ✅ 강력한 멤버십 관리 기능
- ✅ 티어별 보상 설정 가능
- ✅ 커뮤니티 기능 내장
- ✅ 다양한 분석 도구

**단점**:
- ❌ 8% 플랫폼 수수료 (높음)
- ❌ 회원가입 필수
- ❌ 일회성 후원 불가
- ❌ 진입 장벽이 높음

**수수료 구조**:
```
Pro 플랜: 8% + 결제수수료 (~3%)
= 약 11% (거래당)
```

**적합한 경우**:
- 정기적으로 콘텐츠/업데이트를 제공하는 경우
- 후원 티어별 혜택을 차별화하고 싶은 경우
- 이미 팬덤이 형성된 경우

---

### 4. GitHub Sponsors - 개발자 특화

<a href="https://github.com/sponsors" target="_blank">GitHub Sponsors 바로가기</a>

**GitHub Sponsors**는 오픈소스 개발자를 위한 후원 플랫폼입니다.

**장점**:
- ✅ **0% 플랫폼 수수료**
- ✅ GitHub와 완벽한 통합
- ✅ 리포지토리에 후원 버튼 자동 표시
- ✅ 신규 가입 12개월 결제 수수료도 면제

**단점**:
- ❌ **한국 수령 불가** (2026년 1월 기준)
- ❌ 지원 국가 제한 (미국, 일본, EU 등 약 30개국)
- ❌ 신청 후 승인 필요

**한국 개발자 대안**:
```
GitHub Sponsors 수령이 불가능하므로,
Ko-fi 또는 Buy Me a Coffee를 함께 사용하세요.
GitHub README에 두 링크를 모두 추가하면 됩니다.
```

---

### 5. Open Collective - 오픈소스 프로젝트

<a href="https://opencollective.com" target="_blank">Open Collective 바로가기</a>

**Open Collective**는 오픈소스 프로젝트의 재정을 투명하게 관리합니다.

**장점**:
- ✅ 완전한 재정 투명성
- ✅ 팀/프로젝트 단위 후원 관리
- ✅ 세금 처리 지원
- ✅ PayPal, Wise 등 다양한 수령 방법

**단점**:
- ❌ 10% 플랫폼 수수료 (높음)
- ❌ 개인보다 프로젝트에 적합
- ❌ 설정이 복잡함

**적합한 경우**:
- 오픈소스 프로젝트 팀
- 수입/지출 투명성이 필요한 경우
- 세금 처리가 복잡한 경우

### 6. Liberapay - 비영리 플랫폼

<a href="https://liberapay.com" target="_blank">Liberapay 바로가기</a>

**Liberapay**는 비영리로 운영되는 정기 후원 플랫폼입니다.

**장점**:
- ✅ **0% 플랫폼 수수료**
- ✅ 비영리 운영으로 신뢰성 높음
- ✅ 오픈소스 (코드 공개)
- ✅ Stripe, PayPal 지원

**단점**:
- ❌ 일회성 후원 기능 약함
- ❌ 인지도가 낮음
- ❌ UI가 다소 구식

**수수료 구조**:
```
플랫폼 수수료 0%
Stripe: 평균 3.2%
PayPal: 평균 5%
```

## 🇰🇷 한국 후원 플랫폼

### 카카오페이 송금

<a href="https://www.kakaopay.com" target="_blank">카카오페이 바로가기</a>

**카카오페이**는 한국에서 가장 간편한 후원 방법입니다.

**장점**:
- ✅ **수수료 0%** (완전 무료)
- ✅ 대부분의 한국인이 사용
- ✅ 친구 등록 없이 송금 가능
- ✅ QR코드로 간편 결제
- ✅ 즉시 입금

**단점**:
- ❌ 한국 사용자만 가능
- ❌ 해외 결제 불가
- ❌ 후원 전용 기능 없음

**송금 링크 만드는 방법**:
1. 카카오톡 앱 > **더보기** 탭
2. 우측 상단 **스캐너 아이콘** 선택
3. 하단 **송금코드** 클릭
4. **링크 복사**하여 앱/웹에 추가

**링크 형식**:
```
송금코드 QR 또는 공유 링크
(금액 미리 설정 가능)
```

## 📋 앱 스토어 정책 (중요!)

### 결론부터: 후원 링크는 허용

| 스토어 | 순수 후원 링크 | 대가성 후원 |
|-------|--------------|-----------|
| **App Store (한국)** | ✅ 허용 | ❌ 인앱결제 필요 |
| **Play Store** | ✅ 허용 | ❌ 인앱결제 필요 |

**순수 후원**과 **대가성 후원**의 차이:

| 구분 | 예시 | 판정 |
|-----|------|-----|
| 순수 후원 | "개발자 응원하기", "커피 한 잔 사주기" | ✅ 외부 링크 OK |
| 대가성 후원 | "후원하면 광고 제거", "후원자 전용 기능" | ❌ 인앱결제 필요 |

### Apple App Store 상세

**한국 (2021년~)**:
- **인앱결제 강제 방지법** 시행으로 외부 결제 링크 허용
- **순수 후원 링크** → 문제없이 추가 가능
- 디지털 콘텐츠 판매 시에만 중계수수료 적용

**미국 (2025년~)**:
- 법원 판결로 외부 결제 링크 전면 허용, 수수료 부과 금지

### Google Play Store 상세

- 외부 후원 링크 **허용**
- 단, "기부", "후원"임을 **명확히 표시**해야 함
- 후원 대가로 앱 기능을 제공하면 정책 위반

## 🛠️ 모바일 앱 구현 방법

### 방법 1: 단순 외부 링크 (권장)

가장 간단하고 안전한 방법입니다.

**Flutter 예시**:
```dart
import 'package:url_launcher/url_launcher.dart';

void openDonationLink() async {
  final Uri url = Uri.parse('https://ko-fi.com/myusername');
  if (await canLaunchUrl(url)) {
    await launchUrl(url, mode: LaunchMode.externalApplication);
  }
}
```

**React Native 예시**:
```javascript
import { Linking } from 'react-native';

const openDonation = () => {
  Linking.openURL('https://ko-fi.com/myusername');
};
```

**Swift (iOS) 예시**:
```swift
if let url = URL(string: "https://ko-fi.com/myusername") {
    UIApplication.shared.open(url)
}
```

**Kotlin (Android) 예시**:
```kotlin
val intent = Intent(Intent.ACTION_VIEW,
    Uri.parse("https://ko-fi.com/myusername"))
startActivity(intent)
```

## 🎯 마무리

모바일 앱에 후원 링크를 추가하는 것은 생각보다 간단합니다.

**핵심 정리**:

1. **글로벌 사용자** → **Ko-fi** (0% 수수료, 최저)
2. **한국 사용자** → **카카오페이** (0% 수수료, 즉시 입금)
3. **오픈소스 프로젝트** → Ko-fi + Open Collective
4. **정기 후원** → Patreon (수수료 높지만 기능 풍부)

무료 앱을 만드는 개발자라면, 후원 링크 하나 추가해보세요. <br />
생각보다 많은 사용자들이 개발자의 노력에 감사하고 있습니다!
