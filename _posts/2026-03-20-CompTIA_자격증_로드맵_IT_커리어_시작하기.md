안녕하세요. 수수입니다.

IT 분야로 취업·이직하고 싶은데 **어떤 자격증부터 따야 할지** 모르겠다면, CompTIA부터 알아보세요. 전 세계에서 가장 널리 인정받는 **벤더 중립(vendor-neutral) IT 자격증**이고, 미국 국방부(DoD)에서도 공식 인정하는 자격증입니다.

이 글은 CompTIA 자격증의 **전체 그림**을 보여주는 1편입니다. 어떤 자격증이 있는지, 어떤 순서로 따야 하는지, 시험은 어떻게 보는지까지 한 번에 정리했습니다.

## 30초 핵심 요약

- **CompTIA**는 벤더 중립 IT 자격증 — AWS, Cisco 등 특정 회사에 묶이지 않음
- **추천 순서**: A+ → Network+ → Security+ → 전문 분야 선택
- **시험 방식**: Pearson VUE 시험 센터 또는 온라인(자택) 응시
- **유효 기간**: 3년마다 갱신 필요 (CEU 제출 또는 상위 자격증 취득)
- **비용**: 시험당 $265~$530 (자격증에 따라 다름)

## CompTIA란?

**CompTIA(Computing Technology Industry Association)**는 1982년에 설립된 비영리 IT 업계 단체입니다.

| 항목 | 내용 |
|------|------|
| 설립 | 1982년 |
| 본사 | 미국 일리노이주 |
| 자격증 발급 수 | 전 세계 350만 건 이상 |
| 핵심 특징 | 벤더 중립 (특정 회사 제품에 종속되지 않음) |
| 인정 기관 | 미국 국방부(DoD), ISO 17024 인증 |

**벤더 중립이 왜 중요한가?**

Cisco CCNA는 Cisco 장비, AWS SAA는 Amazon 클라우드에 특화되어 있습니다. CompTIA는 **특정 회사에 묶이지 않는 범용 지식**을 검증하기 때문에, 어떤 환경에서든 통하는 기본기를 증명합니다.

## 자격증 전체 로드맵

CompTIA 자격증은 크게 **4단계**로 나뉩니다.

### 1단계: Core (입문)

| 자격증 | 대상 | 핵심 내용 | 시험 코드 | 비용 |
|--------|------|-----------|-----------|------|
| **ITF+ (IT Fundamentals)** | IT 완전 초보 | IT 기초 개념, 용어 | FC0-U71 | $157 |
| **A+** | IT 지원·헬프데스크 | 하드웨어, OS, 네트워크, 보안 기초 | 220-1101 / 220-1102 | $530 (2과목) |

- **ITF+**: IT가 뭔지 아예 모르는 분에게 추천. 필수는 아님
- **A+**: IT 커리어의 **사실상 출발점**. 헬프데스크·IT 지원 취업에 필수

### 2단계: Infrastructure (인프라)

| 자격증 | 대상 | 핵심 내용 | 시험 코드 | 비용 |
|--------|------|-----------|-----------|------|
| **Network+** | 네트워크 관리자 | 네트워크 설계·운영·문제 해결 | N10-009 | $369 |
| **Server+** | 서버 관리자 | 서버 하드웨어·가상화 | SK0-005 | $369 |
| **Linux+** | 리눅스 관리자 | 리눅스 시스템 관리 | XK0-005 | $369 |
| **Cloud+** | 클라우드 엔지니어 | 클라우드 인프라 배포·보안 | CV0-004 | $369 |

- **Network+**: A+ 다음으로 가장 많이 따는 자격증. 모든 IT 분야의 기반
- 나머지는 관심 분야에 따라 선택

### 3단계: Cybersecurity (보안)

| 자격증 | 대상 | 핵심 내용 | 시험 코드 | 비용 |
|--------|------|-----------|-----------|------|
| **Security+** | 보안 입문자 | 위협, 암호화, 접근 제어, 컴플라이언스 | SY0-701 | $425 |
| **CySA+** | 보안 분석가 | 위협 탐지, 분석, 대응 | CS0-003 | $425 |
| **PenTest+** | 모의 해킹 | 취약점 발견, 침투 테스트 | PT0-002 | $425 |

- **Security+**: DoD 8570 인정 자격증. 미국 정부·군 관련 IT 직무에 필수
- **CySA+ vs PenTest+**: 방어(Blue Team) vs 공격(Red Team) 중 택 1

### 4단계: Expert (전문가)

| 자격증 | 대상 | 핵심 내용 | 시험 코드 | 비용 |
|--------|------|-----------|-----------|------|
| **SecurityX (CASP+)** | 시니어 보안 전문가 | 엔터프라이즈 보안 아키텍처 | CAS-005 | $509 |
| **DataX** | 데이터 전문가 | 데이터 분석·거버넌스 | DY0-001 | $369 |

## 나에게 맞는 경로 찾기

### IT 비전공자·처음 시작하는 분

```
ITF+ (선택) → A+ → Network+ → Security+
```

가장 안전한 경로입니다. 기본기를 탄탄하게 쌓고 보안까지 확장합니다.

### IT 지원·헬프데스크 취업 목표

```
A+ → Network+
```

이 두 개면 IT 지원 직무에 충분합니다. 경력 쌓으면서 추가 자격증을 고려하세요.

### 사이버보안 커리어 목표

```
A+ → Network+ → Security+ → CySA+ 또는 PenTest+
```

Security+ 이후 방어(CySA+)와 공격(PenTest+) 중 관심 분야를 선택합니다.

### 이미 IT 경력이 있는 분

```
Security+ 바로 응시 → CySA+ / PenTest+ → SecurityX
```

A+, Network+ 수준의 지식이 이미 있다면 Security+부터 시작해도 됩니다.

## 시험 절차 — 접수부터 합격까지

### Step 1. CompTIA 계정 만들기

[CompTIA 공식 사이트](https://www.comptia.org/)에서 계정을 만듭니다.

- **반드시 신분증과 동일한 이름**으로 가입하세요
- 이름이 다르면 시험 당일 입장이 거부됩니다

### Step 2. 바우처 구매

시험 비용을 결제하는 방법은 두 가지입니다:

| 방법 | 설명 | 팁 |
|------|------|----|
| **CompTIA 직접 구매** | 공식 사이트에서 바로 결제 | 정가 |
| **할인 바우처** | 공인 리셀러에서 구매 | 10~15% 할인 가능 |

> 💡 **학생 할인**: .edu 이메일이 있으면 CompTIA Academic Store에서 **최대 50% 할인** 가능

### Step 3. Pearson VUE에서 시험 예약

CompTIA의 모든 시험은 **Pearson VUE**를 통해 응시합니다.

1. CompTIA 계정에서 "시험 예약" 클릭
2. Pearson VUE 사이트로 이동
3. 시험 선택 (예: SY0-701)
4. **응시 방법 선택**:

| 방법 | 장소 | 장점 | 단점 |
|------|------|------|------|
| **시험 센터** | Pearson VUE 공인 센터 | 안정적 환경, 집중 가능 | 센터까지 이동 필요 |
| **온라인 (OnVUE)** | 자택·사무실 | 어디서든 응시 | 환경 조건 까다로움 |

5. 날짜·시간 선택
6. 결제 또는 바우처 코드 입력
7. 확인 이메일 수신

> ⚠️ **24시간 전까지** 무료 일정 변경 가능

### Step 4. 시험 당일

#### 시험 센터 응시 시

| 순서 | 할 일 |
|------|-------|
| 1 | 시험 시작 **15분 전**까지 도착 |
| 2 | **신분증 2개** 제시 (여권, 운전면허 등) |
| 3 | 소지품 보관 (휴대폰, 시계, 지갑 등 반입 불가) |
| 4 | 화이트보드 또는 메모 용지 수령 |
| 5 | 시험 시작 |

#### 온라인 응시 시 (OnVUE)

| 조건 | 세부 사항 |
|------|-----------|
| 방 | 혼자 있는 조용한 공간 |
| 책상 | 모니터·키보드 외 모든 물건 치우기 |
| 카메라 | 웹캠으로 방 360도 촬영 필요 |
| 인터넷 | 유선 또는 안정적 Wi-Fi |
| 금지 | 이어폰, 듀얼 모니터, 메모지 사용 불가 |

> 💡 **시험 전날** 반드시 Pearson VUE 시스템 점검(System Test)을 실행하세요

### Step 5. 시험 형식

| 항목 | A+ (Core 1) | A+ (Core 2) | Network+ | Security+ |
|------|-------------|-------------|----------|-----------|
| 문항 수 | 최대 90문제 | 최대 90문제 | 최대 90문제 | 최대 90문제 |
| 시간 | 90분 | 90분 | 90분 | 90분 |
| 합격 점수 | 675/900 | 700/900 | 720/900 | 750/900 |
| 문제 유형 | 객관식 + PBQ | 객관식 + PBQ | 객관식 + PBQ | 객관식 + PBQ |

**PBQ(Performance-Based Questions)**란?

단순 객관식이 아니라 **가상 환경에서 실제 작업**을 수행하는 문제입니다. 예를 들어:
- 네트워크 토폴로지 구성
- 방화벽 규칙 설정
- 명령 프롬프트에서 문제 해결

보통 시험당 **3~5개**의 PBQ가 출제됩니다.

> 💡 **팁**: PBQ는 시간이 오래 걸리므로, 먼저 객관식을 풀고 PBQ를 나중에 돌아와서 푸는 전략을 추천합니다

### Step 6. 결과 확인

- **시험 센터**: 시험 종료 즉시 화면에 합격/불합격 표시
- **온라인**: 시험 종료 후 수 분 내 결과 확인
- **공식 점수**: CompTIA 계정에서 **1~5영업일** 내 확인 가능
- **디지털 배지**: Credly를 통해 발급 — LinkedIn에 추가 가능

## 자격증 갱신 — 3년마다 해야 합니다

CompTIA 자격증은 **3년 유효**입니다. 갱신하지 않으면 만료됩니다.

### 갱신 방법

| 방법 | 설명 |
|------|------|
| **CEU 제출** | 교육·활동으로 필요 학점 채우기 |
| **상위 자격증 취득** | 상위 자격증을 따면 하위 자격증도 자동 갱신 |
| **시험 재응시** | 최신 버전 시험을 다시 통과 |

### 자격증별 필요 CEU

| 자격증 | 필요 CEU (3년) | 갱신 비용 |
|--------|---------------|-----------|
| A+ | 20 CEU | $75 |
| Network+ | 30 CEU | $75 |
| Security+ | 50 CEU | $75 |
| CySA+ | 60 CEU | $75 |
| PenTest+ | 60 CEU | $75 |
| SecurityX | 75 CEU | $75 |

### CEU를 얻는 방법

- 온라인 강의·웨비나 수강
- IT 관련 컨퍼런스 참석
- 상위 자격증 또는 타사 자격증 취득
- IT 관련 글·논문 발행
- IT 업무 경력 (연 3 CEU)

> 💡 **가장 쉬운 방법**: Security+를 따면 A+와 Network+가 자동 갱신됩니다

## CompTIA vs 다른 자격증

| 비교 항목 | CompTIA | Cisco (CCNA) | AWS (SAA) | ISC² (CISSP) |
|-----------|---------|-------------|-----------|-------------|
| 벤더 중립 | ✅ | ❌ Cisco 전용 | ❌ AWS 전용 | ✅ |
| 입문 난이도 | 낮음 | 중간 | 중간 | 높음 |
| 경력 요구 | 없음 | 없음 | 없음 | 5년 |
| 비용 | $265~$530 | $330 | $300 | $749 |
| 유효 기간 | 3년 | 3년 | 3년 | 3년 |
| 추천 대상 | IT 입문자 | 네트워크 전문가 | 클라우드 엔지니어 | 시니어 보안 전문가 |

**결론**: IT를 처음 시작한다면 CompTIA가 진입 장벽이 가장 낮고, 이후 전문 자격증으로 확장하기 좋습니다.
