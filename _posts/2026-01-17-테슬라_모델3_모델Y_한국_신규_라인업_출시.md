## 모델Y L (6인승 롱바디) - 한국 출시 임박!

### 한국 인증 완료

테슬라코리아가 **모델Y L** (6인승 롱바디)의 **배출가스·소음 인증**을 완료했습니다. 국내 출시가 임박했다는 신호입니다!

### 모델Y L 주요 스펙

| 항목 | 내용 |
|------|------|
| 시트 구성 | **2+2+2 (6인승)** |
| 휠베이스 | 기존 대비 **+150mm** 연장 |
| 전고 | **1,668mm** (기존 대비 +43mm) |
| 주행거리 (상온) | 복합 **553km** (도심 568km, 고속 535km) |
| 주행거리 (저온) | 복합 **454km** (도심 423km, 고속 493km) |
| 구동방식 | **롱레인지 AWD** (듀얼 모터) |
| 출력 | **456마력** |
| 0-100km/h | **4.5초** |
| 배터리 | **82kWh** |
| 공차중량 | **2,088kg** (기존 1,921kg) |
| 생산지 | 중국 상하이 기가팩토리 |

### 모델Y L 실내 특징

| 항목 | 내용 |
|------|------|
| 2열 시트 | **독립 시트** (캡틴 시트) |

| 3열 기능 | 전동 조절, **열선** |
| 2열/3열 폴딩 | **전동 폴딩** 지원 |

### 확인 사항

1. **생산지**: 현재 한국 출시 모델은 중국 상하이 기가팩토리 생산
2. **FSD 가용성**: 중국산 모델은 FSD(완전자율주행) 기능 제한 가능
3. **보조금 확인**: 거주 지역 지자체 보조금 금액 확인
4. **출고 시기**: 현재 2~3개월 대기

6. **6인승 필요 시**: 모델Y L 출시 대기 권장
