## 1. 필랑트란?

### 차명의 의미

**필랑트(Filante)**는 1956년 르노가 공개한 1인승 초고속 레코드카 **'에투알 필랑트(Étoile Filante)'**에서 유래했습니다.

| 항목 | 내용 |
|------|------|
| **원래 이름** | Étoile Filante (에투알 필랑트) |
| **의미** | 프랑스어로 "별똥별" |
| **원조 차량** | 1956년 시속 300km 돌파한 레코드카 |
| **특징** | 항공기 설계 접목 |

### 포지셔닝

| 항목 | 내용 |
|------|------|
| **세그먼트** | E세그먼트 (준대형) |
| **차종** | 크로스오버 SUV |
| **대체 모델** | SM6, SM7 |
| **의미** | 르노 최초의 준대형 SUV |

## 2. 가격 및 트림

### 가격표

| 트림 | 가격 (세제혜택 적용) |
|------|---------------------|
| **테크노** | 4,331만원 |
| **아이코닉** | 4,696만원 |
| **에스프리 알핀** | 4,971만원 |
| **에스프리 알핀 1955** | 5,218만9천원 (1,955대 한정) |

*개별소비세 인하 및 친환경차 세제 혜택 적용 기준*

### 트림별 주요 차이

**테크노 (기본형)**
- 기본 사양 충실
- 3분기 출시 예정

**아이코닉 (추천)**
- 대부분의 편의사양 포함
- 가성비 최적 트림

**에스프리 알핀**
- 알핀 전용 디자인 요소
- 프리미엄 옵션 패키지

**에스프리 알핀 1955 (한정판)**
- 1,955대 한정 생산
- 런칭 에디션 전용 사양

## 5. 인테리어

### 프리미엄 테크 라운지

필랑트의 실내 콘셉트는 **"프리미엄 테크 라운지"**입니다.

**퍼스트 클래스 라운지 시트:**
- 헤드레스트 일체형 디자인
- 친환경 나파 인조 가죽 (전 트림)
- 8방향 전동 조절
- 열선 + 통풍 기능
- 허리 지지 조절 + 메모리 기능

| **동승석** | 12.3인치 | 스트리밍, 영화, 게임 |

**지원 기능:**

- **긴급 조향 보조(ESA)**
- **후석 승객 알림** (레이더 타입)
- 전방 충돌 경고 및 긴급 제동
- 차선 이탈 경고/유지 보조
- 사각지대 감지
- 어댑티브 크루즈 컨트롤
